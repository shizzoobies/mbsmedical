# MBS Medical Starter Website

This ZIP contains a polished static starter website for MBS Medical.

## What's included
- Multi-page HTML website
- Shared CSS and JS
- Conversion-focused homepage
- Service pages for Primary Care, Weight Loss, and TRT
- Contact page
- Placeholder logo SVG
- WordPress and Elementor planning notes

## Best use case
Use this starter to:
1. Review and refine the content direction
2. Rebuild the sections inside WordPress with Elementor
3. Expand the architecture into future provider, condition, location, and brick-and-mortar pages

## Important notes
- Forms are for layout only and are not connected to a backend
- Contact info is placeholder content
- Replace all treatment, legal, and compliance copy with your final approved language
- Review all service claims with a licensed clinician and legal/compliance advisor before launch
